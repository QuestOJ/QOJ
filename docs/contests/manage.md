# 2.1 比赛管理

未报名比赛的管理员和该场比赛管理者拥有对比赛的管理权限

### 赛制
- OI: 有部分分，赛时只提供样例反馈和样例排名，赛后统一评测
- IOI: 有部分分，赛时提供完整的实时反馈和排名，但赛时无法查看数据
- ACM: 无部分分，赛时提供完整的实时反馈和排名

### Rating 控制
- ABC 800/2000: 新手场，新用户 `Performance` 为 800，最高不超过 2000
- ARC 1000/2800: 普通场，新用户 `Performance` 为 1000，最高不超过 2800
- AGC 1200/INF: 高级场，新用户 `Performance` 为 1200，无最高限制

### 版本控制
- 1: 排行榜显示方式为 ACM 赛制
- 2: 排行榜显示方式为 OI/IOI 赛制

### 实时答疑
选手可在比赛期间对试题发出提问，管理员可在大后台中回答选手问题，发布比赛公告，查看选手真实分数
